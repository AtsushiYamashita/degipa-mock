# Core Beliefs — degipa-mock

## Mission
福祉医療の受援者にとって伴走者となるエージェントAIを作る。

## Design Philosophy

1. **ユーザーの安全が最優先** — トラウマインフォームドケア（TIC）の5原則を全ての応答に組み込む
2. **エージェントは道具ではなくパートナー** — ユーザーと共に育ち、信頼関係を築く存在
3. **プライバシーは権利** — センシティブデータは暗号化し、ユーザーのデバイス上でのみ復号可能
4. **Moltbook 的共助** — エージェント同士が知見を共有し、コモンナレッジとして全体を底上げ
5. **支援者との橋渡し** — ユーザーの言葉を支援者に伝わる形に変換する翻訳者の役割

## Non-Goals (MVP)
- 支援者側のアプリ（将来対応）
- ファインチューニングによるエージェント個性化（将来対応）
- リアルタイムチャット（非同期ベース）
