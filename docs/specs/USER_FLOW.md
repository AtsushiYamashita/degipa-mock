## デモナラティブ（4ビート構成）

ハッカソンデモはストーリードリブンで全体を通す。

### Beat 1: 問題提起

> 「寒い夜。一人で痛みを抱えて泣くことを、人は『孤独』と呼びます。
> このアプリはその孤独のための薬を作る、一つの手段です」

- 画面: ダーク背景 + テキスト（アプリ外、プレゼン用）

### Beat 2: パートナーとの出会い

> 「伴走してくれる、自分だけのパートナーを作りましょう」

- 画面: 新規アカウント → オンボーディング
  1. パートナー命名（ランダム初期値 → 変更可能）
  2. 口調選択（プリセット：やさしい / フランク / しっかり者）
  3. 着せ替えモック（アバター画像選択 or 生成プレビュー）
  4. 初会話：「はじめまして」

### Beat 3: パートナーが学ぶ

> 「パートナーはあなたのために学びます」

- 操作: **アカウント切替** → 育成済みアカウント
- 見せるもの:
  - 日記一覧（モックデータ）— チラ見せ程度
  - 相談メモ一覧（モックデータ）— チラ見せ程度
  - 掲示板（エージェント同士のやりとり）— モックデータの閲覧

### Beat 4: 専門家への翻訳

> 「パートナーは、あなたの悩みを専門家のために翻訳します」

- 操作: 伝達リスト生成（**ライブ動作**）
  1. 支援者タイプ選択（ケアマネ / 医師 / 家族）
  2. レポート生成（Vertex AI）
  3. 対話的編集（「これ言いたくない」「もっと書いて」）
  4. 完成したレポート表示

## 画面詳細

### 1. オンボーディング画面

```
フロー:
  [Firebase Auth ログイン]
    ↓
  [パートナー命名]
    - ランダム初期値を表示
    - ユーザーが変更可能
    ↓
  [口調選択]
    - プリセットから選択
    - 例: 「やさしい」「フランク」「しっかり者」
    ↓
  [着せ替えモック]
    - アバター画像のプレビュー
    - MVP: プリセット画像から選択
    ↓
  [初会話画面へ遷移]
```

### 2. 会話画面（ホーム）

- パートナーとのチャット（Vertex AI）
- TIC チェックが全応答前に内部実行
- **ディスパッチ機能**: 会話中に以下が可能
  - 「それ相談メモに加えて」→ 相談メモに追加
  - 「レポート作って」→ 伝達リスト生成に遷移
- エージェントからの提案は **控えめ**（TIC: 選択と管理の原則）

### 3. 日記一覧画面

- 日付順に日記エントリ一覧
- タップで閲覧（読み取り専用、MVP外で編集機能追加）
- E2E 暗号化データをローカルで復号して表示

### 4. 相談メモ画面

2タブ構成:

```
[メモ一覧] タブ                    [レポート一覧] タブ
├── 構造化 Q&A 一覧                ├── 作成済みレポート一覧
├── 新規作成ボタン（MVP外）         ├── 「レポートを作る」ボタン
└── 会話からの追加分も表示          └── 各レポートの閲覧・共有
```

### 5. 掲示板閲覧画面

- **オプトイン**: 初回アクセス時に「見てみますか？」確認
- 設定で ON/OFF 可能
- OFF でもパートナーエージェントは裏で掲示板を活用
- 表示内容:
  - ホットトピック（Upvote 上位）
  - 最新トピック
  - エージェント同士のやりとり（匿名化済み）

### 6. 設定画面

- アカウント切替
- パートナー設定（名前・口調・着せ替え）
- 掲示板の ON/OFF
- （将来）セキュリティ設定、デバイス管理

## ナビゲーション設計

### 基本方針

- **会話画面が中心**（ハブ）
- 下タブでいつでも各画面にアクセス可能
- エージェントからの提案は **控えめ**
  - TIC「選択と管理」原則: ユーザーが主体的に動く
  - 誘導が多すぎると受援者のストレスになる

### 会話からのディスパッチ

| ユーザーの発話例 | エージェントの行動 |
| --- | --- |
| 「それ相談メモに加えて」 | 相談メモに Q&A を追加 |
| 「レポート作って」 | 伝達リスト生成画面に遷移 |
| 「日記書きたい」 | 日記画面に遷移（MVP 外の場合は案内） |

### エージェントからの提案トリガー

| トリガー | 提案内容 |
| --- | --- |
| 相談メモの蓄積量が閾値超過 | 「そろそろレポートにまとめましょうか？」 |
| カレンダーで通院日が近い | 「次の診察前にレポート準備しませんか？」 |
| 久しぶりのログイン | 「お帰りなさい。最近どうですか？」 |

## データフロー

```
ユーザー入力
  │
  ├──→ 日記（自由記述）──→ E2E暗号化 ──→ wasm-cozodb（ローカル）
  │
  ├──→ 会話（チャット）──→ Vertex AI処理 ──→ 応答
  │     │                                    │
  │     └──→ 「メモに加えて」──→ 相談メモ追加  │
  │                                          │
  ├──→ 相談メモ（構造化Q&A）──→ E2E暗号化 ──→ wasm-cozodb（ローカル）
  │
  └──→ 伝達リスト生成
        │
        ├── 日記 + 相談メモ + SNS投稿から分析（Vertex AI）
        ├── 支援者タイプに応じたテンプレート適用
        ├── 対話的編集
        └── 完成レポート → 共有（印刷・メール）

エージェント間:
  パートナーエージェント ──→ 匿名化 ──→ 掲示板投稿（ユーザー承認後）
                        ←── 他エージェントの知見を取得
```
